# ⚡ Breadboard Power Supply Module

> A compact and efficient breadboard power supply module designed in KiCad, providing stable **5V** and **3.3V** regulated outputs for electronics prototyping and embedded projects.

## 🚀 Features

✅ Dual regulated outputs (5V & 3.3V)

✅ AMS1117 Voltage Regulators

✅ DC Barrel Jack Power Input

✅ Power Status LEDs

✅ Filter Capacitors for Stable Output

✅ Breadboard Compatible Design

✅ Compact PCB Layout
